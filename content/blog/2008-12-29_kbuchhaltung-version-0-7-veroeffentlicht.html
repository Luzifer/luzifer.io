Hierbei handelt es sich in erster Linie um ein BugFix-Release. Das heißt, dass diese Version nur eine einzige Neuerung mitbringt und ansonsten nur zum Beheben bestehender Probleme dient. Hier die Änderungen:
  • 0000018: BugFix: Fehler in der Anzeige nach einer frischen Installation Direkt nach dem Installieren wurden diverse Werte falsch dargestellt.
  • 0000019: Farbige Markierung für Planwerte bei Überschreiten der Grenzen Die Planwerte werden jetzt grün bei Unterschreiten oder genauem Erreichen und rot bei Überschreiten angezeigt um ein besseres visuelles Feeback zu bieten.
  • 0000020: BugFix: Konto wird deselektiert wenn eine Buchung gemacht wird Sobald eine Buchung gemacht wurde, ging die graue Markierung des Kontos auf der linken Seite verloren. Diese bleibt jetzt auch bei einer Buchung weiter bestehen.
  • InfoBox angepasst Die InfoBox enthält jetzt die Informationen, die vorher unten auf der Hauptseite zu finden waren.
Beim Update auf diese Version ist - wie auch bei Version 0.6 - kein Datenbankupdate nötig.

Herunter laden kann man sich die aktuelle Version wie üblich als ZIP oder TAR Datei direkt von GitHub.
