--- title: "Problems with adding a blog to technorati" date: "2008-03-30T19:53:00+00:00" slug: "problems-with-adding-a-blog-to-technorati" --- If you have this issue while adding your blog to technorati:
The proxy server received an invalid response from an upstream server. The proxy server could not handle the request POST /account/blogs/claimchoices.

Reason: Error reading from remote server

Additionally, a 404 Not Found error was encountered while trying to use an ErrorDocument to handle the request.

--------------------------------------------------------------------------------

Apache Server atwww.technorati.comPort 80

You might be able to solve this issue by not using a proxy server to claim your blog. On my site this error disappeared just after switching off my Privoxy.
