--- title: "New project: KWebPortfolio" date: "2008-08-30T22:17:00+00:00" slug: "new-project-kwebportfolio" --- Today I had a new idea for a project and then it was started.

This time it is (as usual) a web based project. This project is intended to help you to show your work portfolio. This could be used for example to show your past work to the manager of a company you want to work for.

Until now the project is in earliest development state and not usable. The git-repository is available at github. More news about this project will be posted here and every commit into the project is posted to friendfeed and twitter.
